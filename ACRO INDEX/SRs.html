Software Requirements Specification
AI Based Personalized Learning Platform
A web-based adaptive learning system powered by Artificial Intelligence
Academic Project  ·  2026
Department: Computer Science & Engineering
 ·  Prepared By: Manish Israni
1
Problem Statement
—
Traditional learning systems follow a one-size-fits-all model and ignore individual student needs
—
Students face undetected learning gaps and weak concepts throughout the course
—
Difficulty in finding relevent and suitable study material for specific topics
—
Lack of personalized feedback causes slow learners to fall behind
—
No adaptive mechanism to adjust learning pace for different students
2
Proposed Solution
—
AI based adaptive learning tailored to each student's pace and level
—
Personalized content recommendations using machine learning algorithms
—
Automatic weak concept detection and learning gap analysis
—
Smart quizzes with real-time evaluation and performance tracking
—
Progress reports and visual dashboards for students and teachers
3
Main Features
User Login & Registration
Secure sign-up and login with role-based access control
AI Recommendation System
Suggests videos, notes and quizzes based on student performance
Performance Analysis
Tracks quiz scores and learning history to detect weak areas
Adaptive Learning Path
Automatically adjusts study plan according to student progress
Quiz System
Auto-evaluated quizzes with instant score and feedback display
Report Generation
Detailed progress reports for students, teachers and admin
4
Technologies Used
—
Frontend: React.js — responsive user interface
—
Backend: Node.js — server-side logic and APIs
—
Database: MongoDB / Firebase — data storage
—
AI Module: Python-based ML recommendation engine
—
Auth: JWT-based secure authentication
—
Protocol: HTTP / HTTPS communication

6
User Roles
Admin
Manages users, content, reports and overall system activities. Has full access to all modules.
Student
Uses the platform to learn, attempts quizzes, views AI-based recommendations and tracks progress.
Teacher
Uploads study material, monitors student performance and provides feedback on progress.
